Aplikacja JStat i jej obsługa

W rozdziale zostanie opisana obsługa programu JStat. Przedstawione zostaną zagadnienia począwszy od instalacji, pierwszego uruchomienia, skończywszy na omówieniu wszystkich modułów programu.

Instalacja i uruchomienie

JStat jeszcze nie ma własnego instalatora. Instalacja programu ogranicza się do skopiowania wszystkich plików programu do folderu na dysku.

Program jest napisany w całości w języku Java.

Java [Developer Resources for Java Technology, http://java.sun.com z dnia 30 kwietnia 2008] jest nowoczesnym obiektowym językiem rozwijanym przez firmę Sun Microsystems. Kod źródłowy kompilowany jest do kodu pośredniego (tzw. bytecode). Kod ten jest niezależny od komputera (w rozumieniu architektury procesora, systemu operacyjnego itp.). Dzięki temu raz skompilowany program można uruchomić na dowolnym komputerze, na którym jest zainstalowana wirtualna maszyna Javy (Java Virtual Machine). W chwili obecnej oprogramowanie Java można pobrać dla następujących systemów:

  • Windows XP/Vista/2000/2003 (wersje 32. i 64. bitowe),
  • Solaris (wersje dla procesorów 32. i 64. bitowych oraz SPARC),
  • Linux (wersje 32. i 64. bitowe),
  • Apple (OS X).

Oznacza to, że program JStat można uruchomić na każdym z tych systemów.

Pobranie środowiska Java

Pakiet do uruchamiania programów napisanych w Javie dla wszystkich wspieranych systemów można pobrać ze strony http://java.com lub najświeższą wersję z http://java.sun.com/javase/downloads/index.jsp. W przypadku pobierania z drugiej lokalizacji, należy wybrać pozycję oznaczoną Java Runtime Environment (JRE). Wersje JRE służą jedynie do uruchamiania wcześniej przygotowanych programów, natomiast za pomocą wersji JDK (Java Development Kit) można ponadto kompilować programy.
W zależności od wybranego systemu, plik instalacyjny zajmuje od około 15. do 20. megabajtów, a po zainstalowaniu całe środowisko zabiera poniżej 100. megabajtów przestrzeni dyskowej.

Zarówno JRE jak i JDK są produktami całkowicie darmowymi, a niedługo także będą w całości wolnym oraz otwartym oprogramowaniem (open-source).

Uruchomienie aplikacji

Po zainstalowaniu Java Runtime Environment wystarczy kliknąć dwukrotnie na ikonę programu (plik nazywa się jstat.jar). Chwilę później powinno się otworzyć główne okno programu, takie jak na zrzutach ekranowych w rozdziale o interfejsie.

Jeśli zaszłaby potrzeba uruchomienia programu z wiersza poleceń, należy najpierw przejść do katalogu, w którym znajduje się plik jstat.jar, a następnie wydać polecenie: 

java -jar jstat.jar
